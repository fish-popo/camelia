<p align="center">React UI library，Implement functionality with the simplest code</p>

## ✨ 特征

- 🌈 为web应用程序设计的企业级UI
- 📦 一组开箱即用的高质量React组件。
- 🛡 用TypeScript编写，具有可预测的静态类型。
- ⚙️ 众多的精致小工具，宛如开发过程中的得力助手
- 🎨 基于主题定制.

## 🖥 支持环境

- 支持现代浏览器.

- 支持服务端渲染.

- 支持Electron桌面应用.

## 📦 安装

```bash
npm install camellia
```

```bash
yarn add camellia
```

```bash
pnpm add camellia
```

## 🔨 使用

```tsx
import { Button } from "camellia";

export default () => (
  <>
    <Button type="primary">Click Me</Button>
  </>
);
```
